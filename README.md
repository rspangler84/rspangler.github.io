# Writing & Editorial Portfolio

This repository collects writing, editing, layout, and document-design samples from
my professional work in the AEC government contracting industry. 

## Portfolio Organization

- **Writing_Samples** — Curated excerpts with my own notes on what I specifically contributed (writing, editing, layout, graphics)
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Writing_Samples

- **Work_Flow** — A numbered walkthrough (draft, layout, client review, print) showing my document production process
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Work_Flow

- **Project_Sheets** — One-page project summary sheets
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Project_Sheets

- **Documents** — Full source reports, studies, and EIS documents, including two project subfolders (AASHTO_TIG, Baudette-Rainy_River)
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Documents

- **Presentations** — Presentation-format materials
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Presentations

- **Interactive_PDF_and_Database_Interface_Design** — Dam safety case studies and the interactive dam owner toolbox
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Interactive_PDF_and_Database_Interface_Design

- **Public_Outreach** — Public-facing outreach materials
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Public_Outreach

- **Flowcharts** — Process flowcharts
  https://github.com/rspangler84/rspangler84.github.io/tree/main/Flowcharts

- **CD_Labels** — CD label design work
  https://github.com/rspangler84/rspangler84.github.io/tree/main/CD_Labels
